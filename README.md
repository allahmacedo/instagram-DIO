## Como usar

 - Clone o repositório em sua maquina
  ```bash
    git clone https://github.com/allahmacedo/StarbucksPage.git
  ```
  - Acesse o folder do projeto
  ```bash 
    cd instagram-DIO
  ```

  - Inicie o index
  ```bash 
    start index.html
  ```
